# AI Agent Configuration

This file contains configuration and usage instructions for AI agents working on this project.

## btca

When you need up-to-date information about technologies used in this project, ask the user if they'd like you to use btca to research.

**Available resources**: react, reactNative, expo, typescript, zustand, tanstackReactQuery, zod, reactNavigation, moti, jest, eslint

### Usage

```bash
btca ask -r <resource> -q "<question>"
```

Use multiple `-r` flags to query multiple resources at once:

```bash
btca ask -r react -r reactNative -q "How do I integrate React with React Native?"
```
